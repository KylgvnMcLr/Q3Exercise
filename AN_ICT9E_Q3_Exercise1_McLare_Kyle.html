What is Bootstrap?

Bootstrap is a free front-end framewrok for faster and easier web development

A free and open source front-end development framework called Bootstrap is used to build websites and web applications. When using it, you'll need to be aware of a few key global styles and parameters that it uses, all of which are primarily focused on standardizing cross-browser styles. HTML5 is a requirement for using Bootstrap. You'll notice some quirky unfinished design without it, but adding it shouldn't result in any major issues.

Why Bootstrap is used?

Bootstrap was created utilizing a mobile-first approach, in which we first optimized the code for mobile devices and then used CSS media queries to scale up components as needed. As a result, responsive web design is now possible. It enables a website or app to recognize the size and orientation of the visitor's screen and automatically adjust the display.
